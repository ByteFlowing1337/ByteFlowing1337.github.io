抢救软路由的趣事一谈

  • router
  • internet

在 istoreOS 的 web 管理页面的 network > interface 中误删了 lan 配置（原本想把校园网拨号接到移动），导致无法进入 web 管理页面。兜了一圈找恢复办法，AI 推荐我用 USB 重刷系统。最后在 2020 年的论坛帖子（来源：https://www.right.com.cn/forum/thread-4016284-1-1.html）里找到了解法：给软路由插上键盘，盲打两条命令即可恢复： 

rm /etc/config/network
reboot

另有一则趣事：电信 163 骨干网的国际出口对非优化线路全时段都很拥堵，而移动的国际出口线路相对更好。我之前一直以为是学校网关对 Reality 协议做了 QoS（理论上能做到，只是误伤率会比 GFW 高），后来才发现是线路出口的问题。
